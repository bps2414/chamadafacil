## PT-BR

### Descrição de 1 linha

ChamadaFácil — Sistema de chamados para suporte técnico.

### Versão curta

- Desenvolvi um sistema web full-stack para abertura, consulta e gerenciamento de chamados técnicos.
- Implementei painel administrativo autenticado com filtros, status, urgência e respostas ao solicitante.

### Versão média

- Desenvolvi o ChamadaFácil, um help desk full-stack para pequenos negócios, com fluxo público de abertura e consulta de chamados.
- Implementei autenticação administrativa com Supabase Auth e painel protegido para gerenciamento da fila de suporte.
- Modelei tickets, respostas e eventos de rate limit em PostgreSQL, usando migrations e Row Level Security.
- Criei interface responsiva em Next.js, TypeScript e Tailwind CSS com estados de loading, erro, vazio e sucesso.

### Versão forte

- Desenvolvi o ChamadaFácil, um sistema web full-stack de chamados para pequenos negócios, com abertura pública, consulta por código/e-mail e painel administrativo.
- Modelei o banco de dados em PostgreSQL para tickets, respostas e controle básico de abuso em formulários públicos.
- Implementei autenticação com Supabase Auth, proteção de rotas administrativas e checagens server-side antes de mutações sensíveis.
- Configurei Row Level Security, uso server-only da service role key, validação server-side e headers de segurança proporcionais ao escopo do MVP.
- Criei dashboard responsivo com lista de chamados, filtros por status/urgência, tela de detalhe, atualização de status e respostas públicas do operador.
- Preparei documentação bilíngue, estudo de caso, guia de deploy e materiais de portfólio/currículo com linguagem profissional e honesta.

## English

### One-line project description

ChamadaFácil — Support ticket system for technical support.

### Short version

- Built a full-stack web application for creating, tracking, and managing technical support tickets.
- Implemented an authenticated admin dashboard with filters, status management, urgency handling, and requester-facing responses.

### Medium version

- Built ChamadaFácil, a full-stack help desk for small businesses, with public ticket creation and lookup flows.
- Implemented admin authentication with Supabase Auth and a protected dashboard for managing the support queue.
- Modeled tickets, responses, and rate-limit events in PostgreSQL using migrations and Row Level Security.
- Created a responsive interface with Next.js, TypeScript, and Tailwind CSS, including loading, error, empty, and success states.

### Strong version

- Built ChamadaFácil, a full-stack support ticket system for small businesses, with public ticket creation, code/e-mail lookup, and an admin dashboard.
- Modeled the PostgreSQL database for tickets, operator responses, and basic abuse control for public forms.
- Implemented Supabase Auth, protected admin routes, and server-side checks before sensitive mutations.
- Configured Row Level Security, server-only service role usage, server-side validation, and security headers aligned with the MVP scope.
- Created a responsive dashboard with ticket listing, status/urgency filters, detail view, status updates, and public operator responses.
- Prepared bilingual documentation, a case study, deployment guide, and portfolio/resume copy with honest professional positioning.
