## PT-BR

### 1 linha

ChamadaFacil - help desk full-stack para pequenos negocios.

### Curta

- Desenvolvi um sistema web full-stack para abertura, consulta e gerenciamento de chamados.
- Implementei painel admin autenticado com busca, filtros, ordenacao, status, urgencia e respostas ao solicitante.

### Media

- Desenvolvi o ChamadaFacil, um help desk full-stack para pequenos negocios, com abertura publica e consulta por codigo/e-mail.
- Implementei autenticacao administrativa com Supabase Auth e painel protegido para operar a fila de suporte.
- Modelei tickets, respostas e eventos de rate limit em PostgreSQL usando migrations e Row Level Security.
- Criei fila admin responsiva com busca, filtros persistidos na URL, ordenacao e destaque de chamados urgentes/sem resposta.
- Adicionei validacao server-side, rate limit basico e testes unitarios com Vitest para regras puras do fluxo.

### Forte

- Desenvolvi o ChamadaFacil, um sistema full-stack de chamados para pequenos negocios, com abertura publica, consulta por codigo/e-mail e painel administrativo protegido.
- Modelei o banco em PostgreSQL para tickets, respostas publicas e controle basico de abuso em formularios publicos.
- Implementei Supabase Auth, protecao de rotas admin, Server Actions, validacao server-side e uso server-only da service role key.
- Configurei Row Level Security, migrations, constraints e headers de seguranca proporcionais ao escopo do MVP.
- Criei dashboard operacional responsivo com busca, filtros por URL, ordenacao, status, urgencia, destaque de sem resposta e respostas publicas.
- Adicionei testes unitarios para validacoes, normalizacao, filtros admin e regras puras de workflow.
- Preparei documentacao bilingue, case study, checklist de screenshots e textos de portfolio com posicionamento tecnico honesto.

## English

### One line

ChamadaFacil - full-stack help desk for small businesses.

### Short

- Built a full-stack web application for creating, tracking, and managing support tickets.
- Implemented an authenticated admin panel with search, filters, sorting, status, urgency, and requester-facing responses.

### Medium

- Built ChamadaFacil, a full-stack help desk for small businesses, with public ticket creation and code/e-mail lookup.
- Implemented admin authentication with Supabase Auth and a protected panel for operating the support queue.
- Modeled tickets, responses, and rate-limit events in PostgreSQL using migrations and Row Level Security.
- Created a responsive admin queue with URL-persisted search/filters, sorting, and urgent/unanswered highlights.
- Added server-side validation, basic rate limiting, and Vitest unit tests for pure workflow rules.

### Strong

- Built ChamadaFacil, a full-stack support ticket system for small businesses, with public creation, code/e-mail lookup, and protected admin panel.
- Modeled the PostgreSQL database for tickets, public responses, and basic abuse control for public forms.
- Implemented Supabase Auth, admin route protection, Server Actions, server-side validation, and server-only service role usage.
- Configured Row Level Security, migrations, constraints, and security headers aligned with the MVP scope.
- Created a responsive operational dashboard with search, URL filters, sorting, status, urgency, unanswered highlights, and public responses.
- Added unit tests for validations, normalization, admin filters, and pure workflow rules.
- Prepared bilingual documentation, case study, screenshot checklist, and portfolio copy with honest technical positioning.
